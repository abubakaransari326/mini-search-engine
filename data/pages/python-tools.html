Python Parsing Tools

BeautifulSoup can parse HTML and extract text along with hyperlink references. It tolerates minor markup quirks, which helps when student authored pages are less strict than production sites. Pair it with a deliberate tokenizer so your index terms remain predictable.

Regular expressions help clean punctuation and normalize tokens. Keep the regular expression simple: extract contiguous alphanumeric runs, fold case, then filter stop words. Complex Unicode rules are optional unless your corpus requires them.

Python data structures make implementation readable and concise. Lists model postings sequences, dictionaries map terms to auxiliary metadata, and small helper functions prevent duplicated logic between indexing and query processing.

When packaging the project, pin library versions lightly and document how to create a virtual environment. A short requirements file plus two commands to install and run goes a long way toward a polished submission.
